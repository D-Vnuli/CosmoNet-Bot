# 🌌 CosmoNet Bot

Telegram-бот для управления VPN-сервисом на базе **3X-UI**.

CosmoNet связывает Telegram-аккаунт пользователя с его VPN-профилем, показывает состояние подписки, выдаёт конфигурацию подключения и автоматизирует административные задачи. Пользователь управляет доступом прямо в Telegram, а актуальное состояние VPN-профиля бот получает из 3X-UI.

> Проект находится в активной разработке. Оплата Telegram Stars и автоматическая выдача подписки уже подключены. Рублёвый эквайринг будет добавлен отдельно.

## ✨ Возможности

### Для пользователей

- регистрация в локальной базе при первом запуске `/start`;
- просмотр актуального статуса подписки из 3X-UI;
- получение персональной ссылки на VPN-конфигурацию;
- выбор месячного тарифа;
- оплата подписки через Telegram Stars;
- создание нового VPN-профиля после оформления подписки;
- продление существующего профиля без замены конфигурации;
- инструкции по подключению на Android, iOS, Windows и macOS.

### Для администратора

- статистика пользователей бота и VPN-профилей 3X-UI;
- просмотр пользователей с активной и неактивной подпиской;
- проверка соединения с 3X-UI;
- защищённая рассылка текста или одной фотографии;
- выбор аудитории рассылки:
  - все пользователи;
  - пользователи с активной подпиской;
  - пользователи без активной подписки;
- итоговая статистика доставки без сохранения истории рассылок в БД;
- тестирование полного цикла создания и продления подписки.

## 💳 Тарифы

Все тарифы действуют 30 дней и отличаются количеством одновременных подключений.

| Тариф | Подключения | Рубли | Telegram Stars |
|---|---:|---:|---:|
| 🚀 Lite | 1 | 129 ₽ | 79 ⭐ |
| ⚡ Standard | 3 | 199 ₽ | 119 ⭐ |
| 👨‍👩‍👧‍👦 Family | 5 | 279 ₽ | 169 ⭐ |

Стоимость в Stars настраивается через `.env` и может быть изменена без правки кода.

Лимит передаётся в поле `limitIp` клиента 3X-UI. Для его фактического применения на сервере должен быть корректно настроен механизм IP Limit.

## 🧭 Как устроен проект

В системе используются два источника данных:

- **SQLite** хранит пользователей, которые запускали Telegram-бота, а также тестовые и реальные заказы;
- **3X-UI** хранит VPN-профили и является источником актуального состояния подписки.

Telegram ID используется как `email` клиента в 3X-UI. Благодаря этому бот может однозначно найти профиль пользователя, создать его при первой покупке или продлить при повторной.

```text
Telegram-пользователь
        │
        ▼
   CosmoNet Bot
     │       │
     │       └── SQLite: пользователи, заказы и идентификаторы платежей
     │
     └────────── 3X-UI: VPN-профили, срок и статус доступа
```

## 🚀 Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/D-Vnuli/MyProjectPlace.git
cd MyProjectPlace/CosmoNet-Bot
```

### 2. Создайте виртуальное окружение

Windows PowerShell:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Linux или macOS:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте окружение

Создайте `.env` на основе `.env.example`:

```env
BOT_TOKEN=YOUR_BOT_TOKEN_HERE

ADMIN_IDS=123456789

XUI_BASE_URL=http://your-server-ip:panel-port
XUI_API_TOKEN=your_api_token
XUI_SUB_BASE_URL=http://your-server-ip:panel-port
XUI_INBOUND_ID=3

TEST_PAYMENTS_ENABLED=false

STARS_PRICE_LITE=79
STARS_PRICE_STANDARD=119
STARS_PRICE_FAMILY=169

PAY_SUPPORT_CONTACT=@your_username
```

| Переменная | Назначение |
|---|---|
| `BOT_TOKEN` | токен Telegram-бота от BotFather |
| `ADMIN_IDS` | Telegram ID администраторов через запятую |
| `XUI_BASE_URL` | базовый адрес панели 3X-UI без завершающего `/` |
| `XUI_API_TOKEN` | API-токен 3X-UI |
| `XUI_SUB_BASE_URL` | публичный базовый адрес ссылок подписки |
| `XUI_INBOUND_ID` | ID VLESS inbound для новых клиентов |
| `TEST_PAYMENTS_ENABLED` | доступ обычных пользователей к тестовой оплате |
| `STARS_PRICE_LITE` | стоимость Lite в Telegram Stars |
| `STARS_PRICE_STANDARD` | стоимость Standard в Telegram Stars |
| `STARS_PRICE_FAMILY` | стоимость Family в Telegram Stars |
| `PAY_SUPPORT_CONTACT` | контакт для команды `/paysupport` |

Если `XUI_INBOUND_ID` не указан, бот автоматически выберет активный VLESS inbound с наибольшим количеством клиентов.

### 5. Запустите бота

```bash
python bot.py
```

При первом запуске SQLite-база будет создана автоматически в `database/cosmonet.db`.

## ⭐ Оплата Telegram Stars

Для Stars не нужен токен стороннего платёжного провайдера, домен или отдельный webhook. Бот работает через стандартные обновления Telegram:

1. пользователь выбирает тариф;
2. бот создаёт заказ и выставляет счёт в валюте `XTR`;
3. Telegram отправляет запрос предварительной проверки;
4. только после события успешной оплаты бот создаёт или продлевает клиента в 3X-UI;
5. пользователю выдаётся VPN-конфигурация.

Идентификатор платежа Telegram сохраняется в заказе. Повторная доставка одного события не продлевает подписку второй раз. Если Stars уже получены, но 3X-UI временно недоступен, пользователь может повторить выдачу подписки без повторной оплаты.

Команда `/terms` показывает условия покупки, а `/paysupport` — контакт для решения вопросов по платежам. Если `PAY_SUPPORT_CONTACT` не заполнен, бот использует первый ID из `ADMIN_IDS`.

## 🧪 Тестовый режим оплаты

Тестовый режим позволяет проверить весь цикл без платёжного провайдера:

1. пользователь выбирает тариф;
2. создаётся тестовый заказ;
3. пользователь подтверждает тестовую оплату;
4. бот создаёт или обновляет клиента в 3X-UI;
5. пользователю выдаётся ссылка на VPN-конфигурацию.

> ⚠️ Деньги в этом режиме не списываются, но профиль в 3X-UI изменяется по-настоящему: обновляются срок действия, статус и лимит подключений.

Администраторы имеют доступ к тестовому сценарию всегда. Чтобы разрешить его зарегистрированным обычным пользователям, установите:

```env
TEST_PAYMENTS_ENABLED=true
```

На рабочем боте оставляйте значение `false`, чтобы пользователи видели только реальную оплату Stars.

## ✅ Тестирование

```bash
python -B -m unittest discover -s tests -v
```

Тесты проверяют:

- тарифы и цены;
- выставление и проверку счетов Telegram Stars;
- защиту от повторной обработки платежа;
- повторную выдачу подписки после временной ошибки 3X-UI;
- создание и повторную обработку заказов;
- создание и обновление клиентов 3X-UI;
- статистику и пользовательские сегменты;
- рассылку, повторные попытки и частичные ошибки;
- права доступа администратора.

## 🔐 Безопасность

- не публикуйте `.env`, токен Telegram-бота и API-токен 3X-UI;
- не добавляйте локальную SQLite-базу в Git;
- ограничьте доступ к панели 3X-UI и используйте HTTPS;
- регулярно создавайте резервные копии базы и конфигурации панели;
- перед рабочим запуском отключите тестовую оплату для обычных пользователей.

Файлы `.env`, локальные базы, виртуальное окружение, кэши Python и настройки IDE исключены через `.gitignore`.
