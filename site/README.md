# Публичный сайт CosmoNet

Статический сайт для размещения на хостинге SpaceWeb. Серверная часть, VPN-узел и Telegram-бот на хостинг не переносятся.

## Перед публикацией

1. Откройте `config.js` и укажите ссылку на Telegram-бота, контакт поддержки, ФИО самозанятого и ИНН.
2. Проверьте тексты в `documents/`: оферта, политика обработки данных и возвраты должны соответствовать фактическому порядку работы.
3. Загрузите содержимое этой папки в `public_html` сайта `cosmonet.shop`.
4. В панели хостинга включите бесплатный Let's Encrypt для `cosmonet.shop` и `www.cosmonet.shop`.

Оплата подписки доступна через Telegram Stars и YooKassa (карты и СБП).
